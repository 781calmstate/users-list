<h1 align = 'center'>:frowning_man: Users List :page_with_curl:</h1>

This application provides a user table where you can add, delete, and edit any user,
as well as explore the details of each user in the list. The project's objective is to enhance knowledge of React, become familiar with new technologies, as well as learn how to create your RESTful API and connect to MongoDB.
